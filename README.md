# 🎮 My-CLI-Games

Welcome to **My-CLI-Games**! 🚀

This repository is a personal collection of terminal-based (Command Line Interface) games written in **C** and **C++**. I created this project to document my learning journey, practice fundamental programming concepts, and improve my algorithmic thinking skills.

By building these games, I am actively practicing:
* Memory management and pointers
* Data structures (arrays, matrices, vectors)
* Object-Oriented Programming (OOP in C++)
* Game loops and state management
* Algorithm design and basic AI (Minimax concepts)
